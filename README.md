# Evaluación Final Módulo 2: SQL 

Este repositorio contiene la solución a la evaluación final del Módulo 2 de SQL, utilizando la base de datos "SAKILA". 

## Descripción

La evaluación consiste en resolver 24 ejercicios de SQL sobre la base de datos "SAKILA", que simula una tienda de alquiler de películas. Las consultas abarcan desde operaciones básicas hasta el uso de subsoncultas, joins y funciones de agregación.
